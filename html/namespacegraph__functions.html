def graph_functions.generate_home_graphs (   numSamples)

This method takes an int 'n' that will later look for the last 'n' rows in the 'sensors_data' and generate automatically three graphs with that data: one for tempertaure, one for humidity an done for lux levels.

It laters stores them in the HOME path.
